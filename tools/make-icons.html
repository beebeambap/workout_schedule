레슨핏 PWA 아이콘 생성기

아래 세 개의 다운로드 버튼을 각각 클릭해서 PNG를 받아 저장소 icons/ 폴더에 넣어주세요. 브라우저가 연속 다운로드를 차단할 수 있어 한 개씩 받는 방식입니다.

팁: 다운로드한 파일명이 icon-192.png처럼 정확히 떨어지지 않으면 수동으로 이름 바꾸세요. 저장 위치는 저장소 루트의 icons/ 폴더.
192 × 192 (필수)
512 × 512 (필수)
180 × 180 (iOS)
  1. 위 3개 버튼을 차례로 눌러 PNG 받기
  2. 받은 파일을 저장소 icons/ 폴더로 이동
  3. git add icons/ && git commit -m "chore: pwa icons" && git push
  4. GitHub Pages 재배포 후 모바일에서 "홈 화면에 추가"
